
# Reguläre Ausdrücke

### Aufgabe 1

Finde Wörter mit `F` im Text. Führe dazu folgendes Beispiel aus:

    import re

    text = "Es war einmal ein Ferkel, das hatte eine Flöte"
    found = re.findall('(F\w+)[^\w]', text, re.IGNORECASE)
    print(found)
